## General info
A university project, functioning as a movie rental service. In the system, there are three types of users:
### Standard User
* Account creation
* Movie database search
* Movie rental
* Password change
* Account deletion
### Admin User
* Movie addition and removal from the database
### Premium User
* Better prices
## Technologies
* Java (JDBC, Swing)
* SQLite
